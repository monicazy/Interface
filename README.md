此分支为接口自动化测试正则版脚本
----------------------------------------------

1. 脚本执行入口为run.py，直接运行此文件就好，不用修改任何东西
2. conf可配置任意sheet内指定用例
3. test_data存放测试用例
4. test_result存放测试结果，包含生成的html报告、图片文件和打印的日志文件
5. tools存放所有公共类
